# 🌾 Modélisation Prédictive pour l'Agriculture

## 📌 Présentation du Projet
Ce projet de Machine Learning vise à analyser les propriétés chimiques et physiques du sol afin d'entraîner des modèles de classification prédictive pour la recommandation et l'optimisation des cultures agricoles.

## 📁 Structure du Dépôt
* `données/` : Contient le jeu de données des mesures du sol (pH, minéraux, etc.).
* `cahiers/` : Notebooks Jupyter d'analyse exploratoire (EDA) et de modélisation prédictive.
* `actifs/` : Ressources visuelles et illustrations du projet.

## 🛠️ Technologies Utilisées
* **Langage :** Python 3
* **Analyse de données :** Pandas, NumPy
* **Visualisation :** Matplotlib, Seaborn
* **Machine Learning :** Scikit-Learn
